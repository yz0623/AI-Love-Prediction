# AI 恋爱契合度测试项目设计文档

## 项目概述
这是一个基于 Next.js 开发的 AI 恋爱契合度测试应用。

## 技术栈
- **框架**: Next.js 15.1.0
- **语言**: TypeScript
- **样式**: Tailwind CSS
- **字体**: Geist (Sans + Mono)
- **代码规范**: ESLint
- **包管理**: npm/yarn/pnpm

## 目录结构 
.

│ ├── components/ # 组件目录
│ └── pages/ # 页面目录
├── public/ # 静态资源
└── [配置文件]
